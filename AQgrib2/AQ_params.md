# GRIB2 (UA) parameters in the Benelux 2018 AQ experiments output

The following table reports the variables included in each of the output files for both air quality runs related to the Benelux 2018 heatwave cases. 

The output is included in the LUMI folders `/scratch/project_465000527/de_33060_common_data/cases/benelux_heatwave_2018/%case/2018/07/%d/%H/`, where `case` is one of  `AQ_Benelux_750m_grib_archive` and `AQD2_25kmL90`, `d` is a day in `{23,24,25,26,27,28}`, `H` is an hour in `{00,06,12,18}`. Each folder contains six files, one for each of the subsequent six hours.
